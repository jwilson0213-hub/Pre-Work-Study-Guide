HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Heading tags should be used in order from h1 as biggest to h6 as smallest heading.
  • Elements nested inside other elements are called child elements.
  • The "body" element is an example of a semantic HTML element, the element describes its role in the webpage.
