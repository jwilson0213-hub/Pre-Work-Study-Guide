Prework Study Guide

HTML

  • Hypertext Markup Language(HTML), is used to give structure to a webpage.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The body element is an example of a semantic HTML element. in semantic elements the name of the element describes its content and the role it has in a website.
  • Non-semantic elements are also used in HTML. Examples: div and span.
  • Elements that are nested in other elements are called child elements.
  • The element that holds all the children elements is known as the parent element.
  • The heading elements h1, h2, h3, h4, h5, h6: represent the level of heading a given text block represents. h1 is the largest, h6 is the smallest.
  • Headings on a webpage should be in order from h1 to h6. h1, the biggest heading, should only appear once on each page.

CSS

  • Cascading Style Sheets(CSS), is a programming language used by developers to define how webpage content is presented to users.
  • CSS defines the colors, aesthetics, visual layout, and organization of a webpage.
  • Is used to define style rules for specific elements of an HTML page and format them with a variety of style options.
  • There are three ways to style a webpage using CSS including inline CSS, internal CSS, and external CSS.
  • Separation of concerns in web development refers to separating code into different files based on use. It is based on the idea that each section of code should have its own responsibility.
  • CSS Declarations contain: 1. The CSS property being applied, 2. The value of the property.

Git

  • Git is a version control system that lets you manage and keep track of each version of your project.
  • git branch: provides a list of your branches. A * will appear next to the current active branch.
  • git merge [branch]: merge a specified branch's history into the current one.
  • git add [file] adds a file as it looks now to the next commit.
  • git commit -m "[descriptive message]" commit your staged content as a new commit snapshot.
  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.

JavaScript

  • JavaScript is a high-level dynamic programming language that allows user interaction with websites.
  • A variable is a named container that allows us to store data in our code.
  • An array is a single variable that holds a group of data that is usually related.
  • Each item in an array is identified with a unique location number called an index.
  • Control flow is the order in which a computer executes code in a script.
  • Conditional statements and loops can be used to control the order in which code is executed, if it is executed at all.
  • A function is a set of instructions that tells the computer how to perform a certain task.
  • You must call a function by name in order to execute it.
